Stay On Top -
Support Tool

  • Stay on Top is a software that was developed as a team for a client (4th Year UW Student) using VBA
  • The software is intended to help organize and assist the client complete their Final Year Design Project
  • I was a lead developer and designer of the software itself and spent approximately 40 hours on the project
  • The various features of the software include:
    • Dynamic calendar that shows all events inputed, which can be modified and removed
    • Kanban board that allows the client to organize tasks
    • A userform that keeps all meeting notes
    • Dynamic availability chart for the week, allowing the team to determine on which days every member is available (availability can be changed)
  • The client was extremely please and impressed with the final software
